Welcome to Micro Front Ends with Module Federation

This is the home page for the container app. Below you will see a nice little counter which can be incremented by 100. There is a counter service coming from an NPM package which is shared by all the applications here. This shows that we can share state across applications.

Counter Service Value: {{ counterService.counter}}

Dynamic Load Component

In one of our child apps we have exposed a component which works with the counter service. If your click the button below it will dynamically load a component from anohter app and it to the container app.

Before you click the button you might want to watch the network tab. As it will show a network request (one time) to load the component dynamically.

Now that you added the component dynamically. Why not go ahead and and increment the counter. All the counter values on this page should be the same. Thanks to module sharing with module federation.
